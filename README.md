# SCT_WD_2 - Calculator Web Application

## 📌 Project Overview

This project was developed as part of the **SkillCraft Technology Web Development Internship**.

The objective of this task was to build a **fully functional Calculator Web Application** to practice **DOM manipulation, event handling, input parsing, error handling, and keyboard interaction** using JavaScript.

To enhance the user experience beyond a basic calculator, the application includes **two calculator modes**:

* Standard Calculator
* Scientific Calculator

Users can seamlessly switch between the two calculators using an interactive toggle interface, providing a modern and professional user experience.

---

## 🚀 Features

### Standard Calculator

* Basic Arithmetic Operations (+, −, ×, ÷)
* Percentage Calculations (%)
* Clear Display Function (C)
* Delete Last Character (⌫)
* Decimal Number Support
* Real-Time Input Handling
* Error Handling for Invalid Expressions

### Scientific Calculator

* Trigonometric Functions (sin, cos, tan)
* Square Root Function (√)
* Logarithmic Functions (log, ln)
* Power Operations (x², xʸ)
* Mathematical Constants (π, e)
* Parentheses Support
* Clear Display Function (C)
* Delete Last Character (⌫)
* Advanced Expression Evaluation

### User Interface Features

* Modern Glassmorphism Design
* Interactive Calculator Toggle
* Color-Coded Functional Buttons
* Smooth Hover & Click Animations
* Fully Responsive Layout
* Professional UI/UX Design
* Dark Theme Interface

### Keyboard Support

* Numeric Input (0–9)
* Arithmetic Operators
* Decimal Input
* Enter Key for Calculation
* Backspace Key for Delete
* Escape Key for Clear

## 💡 How It Works

1. Users can choose between Standard and Scientific Calculator modes.
2. Button clicks dynamically update the display using DOM manipulation.
3. Mathematical expressions are evaluated when the equals (=) button is pressed.
4. Invalid expressions are handled gracefully through error handling mechanisms.
5. Keyboard input support allows calculations without using the mouse.
6. Scientific mode provides advanced mathematical functions and constants.
7. Smooth transitions and animations create a professional user experience.
